To install dependencies:

```bash
npm install
```

Create a `.env` with the following variables:

```dotenv
NEXT_PUBLIC_BACKEND_BASE_URL=<YOUR_BASE_URL>
```

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
